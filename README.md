
# **NARAYANI FOODS**

Narayani Foods is a dynamic website developed to showcase food products, handle customer contact forms, and manage data through an integrated backend system. The project focuses on clean UI, responsive design, and smooth data handling using PHP/MySQL.

## ğŸ“¦ **Setup / Installation Instructions**

1. Install a local server (XAMPP / WAMP / LAMP).
2. Clone or download this repository.
3. Move the folder to the server directory (**htdocs** for XAMPP).
4. Start **Apache** and **MySQL**.
5. Open **phpMyAdmin** and create a database:

   ```
   narayani_db
   ```
6. Import the SQL file (if included) inside the **database** folder.
7. Update database settings in:

   * `config.php`
   * `config2.php`
8. Open the project in browser:

   ```
   http://localhost/NARAYANIFOODS
   ```
